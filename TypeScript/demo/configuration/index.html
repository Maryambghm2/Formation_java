Le fichier tsconfig.json

TS vérifie si un fichier tsconfig existe et utilise les paramètres qui y sont définis.

Avantage : plus besoin de s'inqueter des problèmes

Comment obtenir le fichier tsconfig

  • Le créer manuellement à la racine du projet et le compléter selon les besoins
  • Utilsier la commange : 
    tsc -v
