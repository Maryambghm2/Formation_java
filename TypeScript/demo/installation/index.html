Installation de TS

  • Installer Node.js
  • Vérifier l'installation de Node.js avec la commande 
    node -v
  • Installer TS en global avec la commande 
    npm install -g typescript
  • Vérifier l'installation de TypeScript avec la commande : 
    tsc -- version

Mon premier script TS

  • Ajouter une balise script avec un src un fichier .js
  • Crée un fichier .ts, puis dans la console exécuter la commande : 
    tsc main.ts

Utilisation de tsc avec l'option -w

  • Avec l'option -w ou --watch, vous pouvez préciser le ou les fichiers à surveiller
  • Utilisez ctrl + c
